## ✨ Features at a Glance

| 🌟 Feature                          | 💬 Description                                                                 |
|------------------------------------|---------------------------------------------------------------------------------|
| 🚦 Multimodal Route Planning        | Combine 🚍 bus, 🚆 train, 🚶 walk & more into one seamless route                |
