# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in UrbanFlow, please report it privately.

**DO NOT open a public GitHub issue.** Instead, email **[INSERT EMAIL]** with:

- A description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact

You should receive an acknowledgement within 48 hours, and a fix timeline within 5 business days.

## Scope

- The backend API (`backend/`)
- The mobile application (`urbanflow_app/`)
- Build and deployment scripts (CI/CD, Docker)

## What to expect

1. We will acknowledge receipt within 48 hours
2. We will investigate and determine severity
3. We will release a fix within a reasonable timeline depending on severity
4. We will credit you in the release notes (unless you request anonymity)
